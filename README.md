# Shop Management System (Java)

This is a Java Swing based Shop Management System that allows shop owners to manage products and perform customer billing with a cart system.  
All sales and product data are stored in a MySQL database.

---

## Features

- Add, update, and delete products
- Product stock management
- Search products dynamically
- Customer billing with cart system
- Automatic total calculation
- Save finalized bills to database
- Sales history storage
- MySQL database integration
https://github.com/Junaed-180/Shop-Management-System-Java/tree/main
---

## Technologies Used

- Java (Swing)
- JDBC (MySQL Connector)
- MySQL Database
- XAMPP / MySQL Server

---

## How to Run

1. Install Java JDK (8 or higher)
2. Install MySQL or XAMPP
3. Start MySQL server
4. Run `ShopApp.java`
